Training & Object Detection:
=== YoLOv5 Project: https://pypi.org/project/yolov5/ ===
Aquarium Dataset: https://medium.com/analytics-vidhya/training-a-custom-object-detection-model-with-yolo-v5-aa9974c07088
Custom Object Detection: https://towardsdatascience.com/how-to-train-a-custom-object-detection-model-with-yolo-v5-917e9ce13208
    - should be able to inference videos and images
    - generate dynamic object detection images or real-time videos
Autonomous Driving Dataset: https://github.com/lhyfst/awesome-autonomous-driving-datasets

Train: 
python train.py --img 640 --batch 16 --epochs 50 --data './data/data.yaml' --weights yolov5s.pt --nosave --cache

Test:
python test.py --weights yolov5s.pt --data './data/data.yaml' --task test
python test.py --weights yolov5x.pt --data './data/data.yaml' --img 640 --iou 0.65
